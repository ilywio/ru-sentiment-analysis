# Russian Sentiment Analysis

NLP project for Russian text sentiment classification using classical ML and Transformer-based approaches.

## Project Overview

The project explores different approaches for sentiment analysis on Russian-language text data, including:

- TF-IDF + classical ML models
- Improved TF-IDF pipelines
- BERT-based models
- Data preprocessing and cleaning
- Exploratory Data Analysis (EDA)

The repository contains scripts for preprocessing, model training, experimentation, and evaluation.
